⚠️ Transparency & Moderation Policy

By default, chat messages are NOT stored on our servers. This is a privacy-first application.

If moderation mode is enabled by the server operator, a notice will be shown and chat logs may be retained for safety review. Admins can disable abusive rooms and view file metadata. All admin actions are logged in the moderation audit trail.
